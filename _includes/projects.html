Projects

A small sample of the open-source public-facing applications I have worked on, as well a few solo projects and pieces of code I'm proud of.

FLIQ - A learning engine-driven website delivering smart movie recommendations to users. As a user interacts with the site through either lightning rounds or searching for specific movies they like/dislike, FLIQ builds a profile for the user's preferences using roughly 2,000 data points. Using a custom statistical inference algorithm, a user can be shown smart suggestions for movies that may suit their preferences
Bid-your-time - A Marketplace for organizing and joining events with limited seating using a bidding system. Get the most out of your free time.
ShareIn - A social network for renting useful items from others in your community. Add things you own and want to share to your profile, then others nearby can search for them. For a fee, people can borrow these items from you.
A Scheme Compiler - Scheme to Assembly x86-64 compiler in Scheme. Compiles a large amount of the Scheme language.
Space invaders on STM32 (ARM Cortex-M4) - Connected and built Space Invader in an Embedded Systems Environment using LCD, Wii Nunchuck, and Speaker
